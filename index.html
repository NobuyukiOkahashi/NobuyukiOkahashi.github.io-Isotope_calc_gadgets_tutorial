Tutorial

Making isotope tracing experiments more convenient.

About

"Isotope Calculation Gadgets" are programs that assists data processing for stable isotope tracing experiments. These programs run on the open platform "Garuda".

Overview

Before isotope tracing experiments, an analysis method must be developed. "Isotope Channel Generator" is a gadget supporting the method development.

Obtained raw data can be processed by "Natural Isotope Subtractor", which subtract the effect of natural isotopes. The processed data is also analyzed and mapped by other gadgets.

Imprementation

Step 1 Download and install Garuda Platform 1.4 beta Pack. Learn more.

Step 2 Download and install Python 3.X (Anaconda 4.X.X).

Reboot your computer before procecding to next step.

Step 3 Download and install Isotope Calculation Gadgets.

Step 4 (Optional) For data mapping, download cooperation gadgets and install following instracution. Learn more.
