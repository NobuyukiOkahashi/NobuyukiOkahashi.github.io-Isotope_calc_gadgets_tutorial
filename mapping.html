Data mapping

Processed data by "Isotope Calculation Gadgets" can be visualized on blank maps by using "Shimadzu Multi-Omics Analysis Gadget Pack" and "VANTED".

Preparation

Obtain natural isotope-subtracted labeling data by using "Natural Isotope Subtractor".

Procedure

Click data to visualize, then click "Discover" Button. Select "Shimadzu MSdata import."

Note:

  • If you can't find "Shimadzu MSdata import", download and install it following instruction (Step 4 in imprementation).

    • Click "Fluxome" button in the pop-up window.

    Click "Flux Data Expression Data" button in the next window.

    The data will be imported in "Shimadzu MSdata import".

    Note:

  • If the screen freezes, save the result file of Natural isotope subtractor and reload it in the newly launched Shimadzu MSdata import.

    • Fill the names of the sample groups in the sample ID row. Check "Sample ID" in the Group box. Click Discover and select "Multiomics Data Mapper".

    "Multiomics Data Mapper" will start up.

    Click "Pathway GML File" and choose a blank map. Then, click the "Launch" button.

    Note:

  • Ready-made blank maps can be obtained from here.

    • Click the "Disocver" button and select "VANTED".

    A map with the data visualized will be displayed.

    Note:

  • The map can be modified as you like.

    • Saved the map as image. Click "File" > "Export" > "Network as Image", then choose an extension.
